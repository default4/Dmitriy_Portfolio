PC Components Analysis Project

This project focuses on an in-depth analysis of datasets related to various PC components. The primary goal is to derive insightful visualizations and predictive models that can shed light on trends, correlations, and patterns concerning PC parts.

PROJECTS

PC Components Analysis Project

June 23, 2023

Goals

  1. Insightful Visualization: Understand and visualize the distribution of various component features, trends over time, and identify correlations between different attributes.
  2. Price Predictions: Create machine learning models to forecast the future prices of PC components, enabling users to make informed decisions in a rapidly changing tech landscape.

Datasets

The dataset contains following columns:

  1. CPU: Contains details about various CPU models, their speeds, and respective prices.
  2. GPU: Lists different GPU models, their VRAM capacities, and prices.
  3. RAM: Details different RAM modules, their sizes in GB, and prices.
  4. Storage (SSD): Contains data about various SSD models, their storage capacities, and prices.
  5. Motherboard, Power Supply, and Cabinets: These datasets offer details about other essential PC components.

Correlation between GPU VRAM and its price

In the visualization illustrating the correlation between GPU VRAM (Video Random Access Memory) and its price:

  • The scatterplot shows a general upward trend, indicating that GPUs with higher VRAM tend to be priced higher. This is expected since more VRAM typically offers better performance, especially for tasks that require a lot of memory, such as gaming or graphic-intensive applications.
  • There are a few GPUs with the same VRAM but different prices. This variation can be attributed to other factors like brand reputation, additional features, GPU architecture, or market demand.

Correlation between CPU Speed and its price

In the visualization illustrating the correlation between CPU speed and its price:

  • The scatterplot shows a general upward trend, indicating that CPUs with higher speeds tend to be priced higher. However, there are several exceptions, especially in the mid-speed range, where CPUs have varying prices.
  • There are a few CPUs with similar speeds but significantly different prices, suggesting that other factors (like brand, additional features, or market demand) also influence the price.

Correlation between RAM and its price

In the visualization illustrating the correlation between RAM size (in GB) and its price:

  • The scatterplot shows a general upward trend, indicating that RAM modules with larger capacities tend to be priced higher. This is expected, as larger RAM modules allow for better multitasking and performance in various tasks.
  • There are several RAM modules with the same size but different prices. This variation can be attributed to other factors, such as brand reputation, RAM speed, type (e.g., DDR3 vs. DDR4), or other features.

Correlation between SSD Storage Capacity and its price

In the visualization illustrating the correlation between SSD storage capacity (in GB) and its price:

  • The scatterplot demonstrates a general upward trend, confirming that SSDs with larger storage capacities are typically priced higher. This relationship is expected since greater storage allows for more data, applications, and files to be stored.
  • There’s a noticeable concentration of SSDs in the lower capacity range (below 500GB), reflecting the popularity and affordability of these sizes.
  • The few data points at higher capacities (above 1000GB or 1TB) also follow the trend, but there are fewer options available in this range.
  • As with other components, there are several SSDs with the same storage capacity but varying prices. This difference can be attributed to factors like brand, read/write speeds, type (e.g., NVMe vs. SATA), and other features.

Correlation between PowerSupply Wattage and its price

In the visualization illustrating the correlation between power supply wattage and its price:

  • The scatterplot demonstrates a general upward trend. This indicates that power supplies with higher wattages, which can support more power-demanding components, tend to be priced higher.
  • There’s a noticeable concentration of power supplies in the mid-wattage range (around 500-700W). This reflects the typical requirements of many mainstream PC builds.
  • Power supplies at the higher wattage range (above 800W) are fewer, and their prices vary more widely. This suggests that other factors, such as efficiency ratings, modularity, or brand reputation, also influence the price in this range.

Price Predictions

Steps for Price Prediction:

  1. Data Preprocessing: Clean the dataset, handle missing values, and prepare features for modeling.
  2. Feature Selection: Identify and select the main features that might influence the CPU price.
  3. Data Splitting: Split the dataset into training and testing sets.
  4. Model Training: Train a regression model on the training data.
  5. Model Evaluation: Evaluate the model’s performance on the testing data.
  6. Prediction: Use the trained model to predict prices for new or unseen CPU data.

First will be used a simple linear regression model to predict CPU prices based on the ‘Speed’ feature.

The trained linear regression model on the training set has the following performance metrics:

  1. Root Mean Squared Error (RMSE): 14,802.62 This indicates that the model’s predictions, on average, deviate by approximately 14,802.62 units from the actual prices.
  2. R-squared (R²): 0.2822 The R² value represents the proportion of the variance in the dependent variable (price) that is predictable from the independent variable (speed). An R² value of 0.2822 suggests that around 28.22% of the variability in the CPU prices can be explained by the model using the speed feature.

The linear regression model’s performance on the testing set is as follows:

  1. Root Mean Squared Error (RMSE): 14,299.86 This means that, on average, the model’s predictions deviate by approximately 14,299.86 units from the actual prices in the testing set.
  2. R-squared (R²): 0.3086 An R² value of 0.3086 on the testing set indicates that the model can explain around 30.86% of the variability in the CPU prices using the speed feature.

Suppose there are three new CPUs with speeds of 2.5 GHz, 3.5 GHz, and 4.5 GHz. We’ll predict their prices using our trained linear regression model.

Using the trained linear regression model, here are the predicted prices for the new CPUs based on their speeds:

  • CPU with 2.5 GHz: Approximately -6,088.35 (This negative prediction is unrealistic and highlights the limitations of the linear model when extrapolating outside the training data range.)
  • CPU with 3.5 GHz: Approximately 12,607.21
  • CPU with 4.5 GHz: Approximately 31,302.76

It’s evident that a simple linear regression model might not be sufficient, especially for data points that fall outside the training data range.

It’s need to try using a more complex regression model to potentially achieve better performance. One such model is the Random Forest Regressor, which is an ensemble learning method. It can capture non-linear relationships and is generally more robust than simple linear regression.

The performance of the Random Forest Regressor on the testing set is as follows:

  1. Root Mean Squared Error (RMSE): 12,651.08 This means that, on average, the model’s predictions deviate by approximately 12,651.08 units from the actual prices in the testing set.
  2. R-squared (R²): 0.4589 The R² value of 0.4589 indicates that the Random Forest model can explain around 45.89% of the variability in the CPU prices using the speed feature.

Comparatively, the Random Forest Regressor shows better performance metrics than the simple linear regression model. The increased R² value and reduced RMSE suggest improved predictions.

Using the trained Random Forest Regressor, here are the predicted prices for the new CPUs based on their speeds:

  • CPU with 2.5 GHz: Approximately 17,797.87
  • CPU with 3.5 GHz: Approximately 7,690.60
  • CPU with 4.5 GHz: Approximately 46,036.76

These predictions appear more realistic and are free from the negative values observed with the linear regression model.

Result

Through this project, we achieved a comprehensive understanding of the PC components market, recognizing significant trends and correlations. The predictive models built offer valuable insights into future price trends, aiding both consumers and businesses in making strategic decisions.
